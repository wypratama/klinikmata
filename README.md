# klininkmata



'/' menampilkan halaman login atau tombol daftar jika belum memiliki akun
'/daftar' menampilkan form pendaftaran setelah selesai pendaftaran otomatis login atau kembali kehalaman login lagi jika susah
'/dashboard' menampilkan data reservasi pasien di hari tersebut
'/daftar-pasien' menampilkan daftar pasien
'/dokter' menampilkan daftar dokter
'/reservasi' menampilkan form reservasi pasien
'/reservasi/:id/menampilkan reservasi berdasar id
/reservasi/:id/lewat' menampilkan reservasi yang sudah lewat disertai dengan form untuk memasukkan diagnosa dan resep
